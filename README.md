# Journey to the Down Under - 澳洲紐西蘭打工度假遊記 (2006)

A travel memoir website documenting a working holiday journey through Australia and New Zealand in 2006.

## About

This is a personal travel blog written in Traditional Chinese, chronicling experiences during a working holiday adventure across Australia and New Zealand. The site contains 14 chapters covering different regions and experiences, from Sydney to Tasmania, from Perth to New Zealand.

## Modernization

This website has been modernized from its original 2006 HTML4 format to modern HTML5 standards:

- ✅ HTML5 semantic structure
- ✅ Responsive design with CSS Grid/Flexbox
- ✅ CSS custom properties (variables)
- ✅ Mobile-friendly viewport
- ✅ Accessibility improvements
- ✅ Modern CSS with proper attribute quoting
- ✅ Replaced deprecated tags (`<font>` → `<span>`)

## Structure

- `index.html` - Home page with chapter navigation
- `ch00.html` - `ch14.html` - Chapter pages (with iframes)
- `ch*_txt*.html` - Chapter content pages
- `preface.html`, `ending.html` - Preface and epilogue
- `style.css` - Main stylesheet
- `images/` - Photo gallery
